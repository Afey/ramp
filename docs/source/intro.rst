
Getting started
===============

Installation
------------

Minimum requirements:
 * Numpy
 * Scipy
 * Pandas
 * Scikit-learn

Recommended packages:
 * Py-tables (for fast persistence)
 * nltk (for various text features)
 * Gensim (for topic modeling)
    
Install::

    pip install ramp

or::

    python setup.py install
